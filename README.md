# CI/CD Pipeline with Docker, GitHub Actions & Azure App Service

A complete end-to-end DevOps project where a Python Flask application is:
- containerized using **Docker**
- pushed to **Docker Hub**
- automatically deployed to **Azure App Service (Linux container)**
- using a **GitHub Actions CI/CD pipeline** with Azure Service Principal authentication

This project demonstrates real-world DevOps workflows suitable for cloud engineer / DevOps fresher roles.
